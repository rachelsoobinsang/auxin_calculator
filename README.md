# +/- Auxin Master Mix Calculator
**Rachel Sang**

## Code Objective
Objective: Given an input of a desired plate concentration and number of plates, return the recipe for the spiking step of plate making.  
NOTE: This calculator does not account for pipetting error. In other words, if you really need 10 plates, input a number larger than 10 (to make extra.)  
Inputs: Desired plate concentration, Number of Plates, Regular v. Lifespan 
Output: Spiking drug master mix recipe


## How to Run Code
First, clone this git repository onto your local computer. You only need to do this once (or if I update the code on GitHub.) 
1. Open the repository in GitHub. Click the green "<> Code" button, and copy the HTTPS or SSH key. 
2. Open your terminal. Copy the following: cd/Desktop.
3. Clone the repository onto your Desktop. Copy the following: git clone (HTTPS/SSH key)  
    Note: Replace (HTTPS/SSH key) with the HTTPS or SSH key you copied earlier.
    
At this stage, you should have a copy of the algorithm on your local computer. Check your desktop, and look for a folder called auxin_calculator to verify.

Once you have confirmed that cloning was successful, we can run our code.
1. Open your terminal. Copy the following: cd ~/Desktop/auxin_calculator
2. Confirm that you are in the auxin_calculator directory. Your terminal should look something like this: (base) rachelsang@wifi-10-44-83-121 auxin_calculator %
3. Install the required packages. Copy the following: pip3 install -r requirements.txt
4. Copy the following into your terminal: python3 auxin_calculator.py

Follow the on-screen instructions to get your drug recipes.

Once you have finished inputting your desired plates, open your auxin_calculator folder on your Home screen.
There should be a new folder inside auxin_calculator called outputs. You can find a spreadsheet of every calculation that you just ran.
